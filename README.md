
# GundamMS-SSD-ChainerCV

## Overview

Recognize Gundam's mobile suits using SSD-300.  
Supported frameworks are Chainer and ChainerCV.

* [Chainer](https://github.com/chainer/chainer)
* [ChainerCV](https://github.com/chainer/chainercv)
